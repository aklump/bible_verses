SUMMARY: A PHP solution for collecting Bible verses into collections, using
external APIs to provide the text.


CONFIGURATION:
* pages/our-father.txt has been provided as an example; you may build as many
  collections (pages) as you'd like
* Create a file inside of pages called, our-father.txt
* The first line of our-father.txt will be the page title
* Then, enter the passages you would like to display one per line into
  our-father.txt
* Visit the homepage/living and you'll see the page

STYLING:
* Modify custom.css as needed; do not modify includes/style.css


API:
* Extend Bible class as needed for correct translation
